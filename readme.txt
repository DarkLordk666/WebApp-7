download XAMPP : https://www.apachefriends.org/download.html
1. เปิด xampp
2. รัน Module Apache และ MySQL กดที่ Actions
3. เมื่อ Module ทั้งสองรันจนเป็นสีเขียวแล้ว ให้กดที่ปุ่ม admin ที่ MySQL โปรแกรมจะขึ้นหน้า phpMyAdmin บน browser

Visual
1. โฟล์เดอร์สำหรับส่วน backend
1.1 ใช้คำสั่ง npm install express cors ที่ terminal
1.2 สร้างไฟล์ app.js

2. โฟล์เดอร์สำหรับส่วน frontend
2.1 สร้างโฟล์เดอร์เพื่อใช้ในการทำงานของ frontend เปิด command prompt เลือกตำแหน่งที่จะติดตั้งให้เรียบร้อยและพิมพ์ npx create-react-app {ชื่อโฟลเดอร์ที่ต้องการ}
2.2 เปิดไฟล์นี้ผ่านแป้นพิมพ์บนวินโดว์ ห้ามเปิดผ่านตัว visual โดยตรงเพราะมันจะแทนที่การทำงานของโฟล์เดอร์ backend
