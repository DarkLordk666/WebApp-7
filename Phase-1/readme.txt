Homepage.html

<!DOCTYPE html>: แถกเปิดส่วนหัวของเอกสาร HTML5 เพื่อระบุเวอร์ชันของ HTML ที่ใช้ในเว็บไซต์.
<html>: แท็กหลักที่ครอบเนื้อหาทั้งหมดของเว็บไซต์.
<head>: เป็นส่วนที่ใช้ในการระบุข้อมูลที่เกี่ยวกับหน้าเว็บ เช่น ชื่อหน้าเว็บ (ใน <title>) และลิงก์ไปยังไฟล์ CSS (ใน <link>).
<title>: กำหนดชื่อของหน้าเว็บที่จะแสดงในแท็บของเบราว์เซอร์.
<link>: ระบุลิงก์ไปยังไฟล์ CSS (style.css) เพื่อให้เว็บไซต์นำไปใช้สไตล์.
<body>: เป็นส่วนที่ใช้ในการเริ่มเนื้อหาหลักของหน้าเว็บ.
<header>: ส่วนหัวของเว็บไซต์ ประกอบด้วยเมนูการนำทาง.
<nav>: ส่วนเมนูการนำทางของเว็บไซต์ ประกอบด้วยรายการเชื่อมโยงไปยังหน้าเว็บอื่น ๆ.
<ul>: แท็กรายการไม่สัดส่วน (unordered list) ใช้สำหรับเรียงลำดับรายการเมนู.
<li>: แท็กรายการของเมนู.
<a>: แท็กลิงก์ (เชื่อมโยง) ไปยังหน้าเว็บอื่น ๆ.
<section>: แท็กสำหรับแบ่งเนื้อหาในหน้าเว็บออกเป็นส่วนย่อย ๆ เช่น เนื้อหาหลักและรายการเกมส์.
<h1>: แท็กสำหรับหัวข้อหลักขนาดใหญ่ เช่น "ยินดีต้อนรับสู่เว็บขายเกมส์."
<p>: แท็กสำหรับข้อความปกติหรือประกาศ เช่น "เรามีสินค้าและบริการที่ดีที่สุดสำหรับคุณ."
<h2>: แท็กสำหรับหัวข้อย่อยขนาดใหญ่ เช่น "รายการเกมส์."
<ul>: แท็กรายการไม่สัดส่วนใช้แสดงรายการของเกมส์.
<li>: แท็กรายการเกมส์แต่ละรายการ.
<a>: แท็กสำหรับลิงก์ไปยังหน้าเกมส์แต่ละเกมส์.
<div>: แท็กสำหรับกล่องรูปภาพและข้อความเกี่ยวกับเกมส์.
<img>: แท็กสำหรับแสดงรูปภาพเกมส์.
<strong>: แท็กสำหรับเน้นข้อความชื่อเกมส์.

โค้ดด้านบนเป็นโค้ด HTML สำหรับหน้าหลักของเว็บไซต์ ซึ่งมีเมนูการนำทางด้านบน หัวข้อหลักและรายการเกมส์ที่แสดงในรูปแบบรายการไม่สัดส่วน 
โดยมีภาพและข้อความเกี่ยวกับเกมส์แต่ละรายการ สามารถแก้ไขเนื้อหาและสไตล์ต่างๆ ของหน้าเว็บได้ตามต้องการโดยแก้ไขไฟล์ style.css 
และเพิ่มหรือลบเนื้อหาใน HTML ตามที่ต้องการ

<!DOCTYPE html>: แถกเปิดส่วนหัวของเอกสาร HTML5 เพื่อระบุเวอร์ชันของ HTML ที่ใช้ในเว็บไซต์.
<html lang="en">: แท็กหลักที่ครอบเนื้อหาทั้งหมดของเว็บไซต์และระบุภาษาเป้าหมายของหน้าเว็บ.
<head>: ส่วนหัวของเว็บไซต์ที่ใช้ในการระบุข้อมูลเพิ่มเติมเกี่ยวกับหน้าเว็บ.
<meta charset="UTF-8">: แท็กที่ระบุชุดอักขระ (charset) เป็น UTF-8 ซึ่งเป็นชุดอักขระที่ใช้กว้างขวางและรองรับตัวอักษรหลากหลาย.
<meta name="viewport" content="width=device-width, initial-scale=1.0">: แท็กที่ระบุการกำหนดค่าการแสดงผลบนอุปกรณ์เคลื่อนที่ เพื่อให้เว็บไซต์แสดงผลได้ถูกต้องบนหน้าจอขนาดต่าง ๆ.
<title>ลงทะเบียน</title>: กำหนดชื่อของหน้าเว็บ.
<link rel="stylesheet" type="text/css" href="style.css">: ลิงก์ไปยังไฟล์ CSS (style.css) เพื่อให้เว็บไซต์นำไปใช้สไตล์.
<link rel="stylesheet" type="text/css" href="register.css">: ลิงก์ไปยังไฟล์ CSS สำหรับหน้าลงทะเบียน (register.css) เพื่อให้เว็บไซต์มีสไตล์พิเศษสำหรับหน้านี้.
<body>: เปิดส่วนเนื้อหาหลักของหน้าเว็บ.
<header>: ส่วนหัวของเว็บไซต์.
<nav>: ส่วนเมนูการนำทางของเว็บไซต์ ในที่นี้มีลิงก์กลับไปหน้าหลัก (Homepage).
<ul>: รายการไม่สัดส่วน (unordered list) ใช้สำหรับเรียงลำดับรายการเมนู.
<li>: รายการของเมนู.
<a>: ลิงก์ไปยังหน้าเว็บอื่น.
<section>: เปิดส่วนเนื้อหาหลักของหน้าลงทะเบียน.
<h1>: หัวข้อหลักขนาดใหญ่สำหรับหน้าลงทะเบียน.
<p>: ข้อความประกาศ.
<form>: แบบฟอร์มสำหรับผู้ใช้ลงทะเบียน.
<label>: ป้ายชื่อข้อมูลที่ให้ผู้ใช้กรอก.
<input>: ช่องใส่ข้อมูล เช่น ชื่อผู้ใช้, รหัสผ่าน, อีเมล.
type="text", type="password", type="email": ระบุประเภทของช่องใส่ข้อมูล.
id: ระบุ ID ของช่องใส่ข้อมูล.
name: ระบุชื่อของช่องใส่ข้อมูล.
pattern: ระบุรูปแบบที่ข้อมูลต้องเป็นไปตาม เช่น รหัสผ่านต้องมีอักขระอย่างน้อย 1 ตัวตัวเลขและ 1 ตัวอักษรพิเศษ # หรือ * และมีอย่างน้อย 10 ตัวอักษร.
required: ระบุว่าข้อมูลจำเป็นต้องกรอก.
title: ระบุข้อความที่จะแสดงหากผู้ใช้กรอกข้อมูลไม่ถูกต้อง.
<input type="submit">: ปุ่มสำหรับส่งแบบฟอร์ม.

หน้าลงทะเบียนนี้มีแบบฟอร์มสำหรับกรอกข้อมูลลงทะเบียน เช่น ชื่อผู้ใช้, รหัสผ่าน, ยืนยันรหัสผ่าน, และอีเมล และมีการกำหนดเงื่อนไขในการกรอกข้อมูลแต่ละช่องเพื่อให้ผู้ใช้กรอกข้อมูลได้อย่างถูกต้องและครบถ้วน

แบบฟอร์มนี้มีช่องใส่ชื่อผู้ใช้ รหัสผ่าน และปุ่มเข้าสู่ระบบสำหรับผู้ใช้กรอกข้อมูลและทำการเข้าสู่ระบบบนเว็บไซต์.

Search.html

<!DOCTYPE html>: ระบุรูปแบบของเอกสาร HTML ว่าเป็น HTML5.

<header>: ส่วนหัวของเว็บไซต์.
<nav>: ส่วนเมนูการนำทางของเว็บไซต์ ในที่นี้มีลิงก์ไปหน้าหลัก (Homepage), เข้าสู่ระบบ (Login), และลงทะเบียน (Register).

<li>: รายการของเมนู.
<a>: ลิงก์ไปยังหน้าเว็บอื่น.

<h1>: หัวข้อหลักขนาดใหญ่สำหรับหน้า "ค้นหาสินค้าและบริการ".
<form>: แบบฟอร์มสำหรับการค้นหาสินค้าและบริการ.
<input>: ช่องใส่ข้อมูลสำหรับกรอกคำค้นหา.
<button>: ปุ่ม "ค้นหา" สำหรับเริ่มกระบวนการค้นหา.
<div id="search-results">: ส่วนที่ใช้แสดงผลลัพธ์การค้นหา.
<h2>ผลลัพธ์จำลอง</h2>: หัวข้อสำหรับแสดงผลลัพธ์การค้นหาจำลอง.
<table>: ตารางสำหรับแสดงผลลัพธ์การค้นหา.
<thead>: ส่วนหัวของตาราง.
<tr>, <th>: แถวและหัวของตาราง.
<tbody>: ส่วนเนื้อหาของตาราง.
มีการใช้วงเล็บเปิดปิด (<!-- และ -->) เพื่อความสะดวกในการเพิ่มข้อมูลจำลองตามความต้องการ.
<script>: ส่วนที่ใช้ในการเพิ่มสคริปต์ JavaScript เพื่อการค้นหาและแสดงผลลัพธ์การค้นหา.

ส่วน JavaScript ที่ถูกเพิ่มไว้ในหน้านี้เป็นตัวอย่างการค้นหาและแสดงผลลัพธ์ที่สร้างขึ้นแบบจำลอง (hardcoded) เพื่อแสดงวิธีการทำงานของการค้นหาบนหน้าเว็บไซต์ของคุณ 
โดยมีฟังก์ชัน performSearch() ที่ใช้ในการค้นหาข้อมูลจากข้อมูลที่มีอยู่และแสดงผลลัพธ์ในตาราง.
ต้องการปรับปรุงเนื้อหาและการค้นหาจริงๆ ให้เชื่อมต่อกับฐานข้อมูลหรือแหล่งข้อมูลจริง ๆ และนำข้อมูลจากฐานข้อมูลมาแสดงผลในตารางแทนการใช้ข้อมูลจำลอง.

style.css

.image-box: คลาส .image-box ถูกใช้ในรายการเกมส์ และได้กำหนดความกว้างของกล่องรูปภาพเป็น 100px และความสูงปรับแต่งอัตโนมัติตามอัตราส่วนของรูปภาพ ที่แสดงอยู่ในกล่องนี้ 
โดยการใช้ display: inline-block; จัดเรียงรูปภาพให้อยู่ในแนวนอนและกำหนดระยะห่างด้านขวาของกล่องรูปภาพเป็น 10px.
.image-box img: กำหนดรูปภาพในกล่องรูปภาพให้มีความกว้างเต็มกับกล่อง .image-box และความสูงปรับแต่งอัตโนมัติตามอัตราส่วนของรูปภาพ 
และใช้ object-fit: cover; เพื่อให้รูปภาพครอบคลุมพื้นที่ในกล่อง ทำให้รูปภาพที่แสดงอยู่ภายในกล่องรูปภาพมีขนาดเหมือนกันและไม่ถูกตัดขอบหรือย่อ/ขยายให้สะท้อนออกจากกล่อง.
CSS อื่น ๆ ที่ในไฟล์ style.css คือการปรับแต่งรูปแบบของหน้าเว็บโดยรวม การตั้งค่าพื้นหลัง ตัวอักษร การจัดรูปแบบเมนู หัวข้อหน้าเว็บ การจัดรูปแบบของรายการเกมส์ และการเพิ่มสไตล์สำหรับกล่องรูปภาพของสมาชิก.
